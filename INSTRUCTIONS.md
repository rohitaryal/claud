# What to use for

## 1. Front-End

- React + Vite: Easy to work with, many tools available around it, easy to learn
- TypeScript: Type-safe and enables faster code writing
- More to add as we go through

## 2. Back-End

- Node: For building and serving front-end during development
- Wrangler: CLI to manage worker projects
- Hono.js: Good support with CloudFlare workers, full power with TypeScript
- More to add as we go through
